# WB → Ozon Price Comparator

Telegram бот для сравнения цен между Wildberries и Ozon. На вход принимает артикул товара с Wildberries, на выходе показывает самый дешевый аналог на Ozon.

## Возможности

- ✅ Парсинг товаров Wildberries по артикулу (через API + Selenium)
- ✅ Поиск аналогов на Ozon по названию и бренду
- ✅ Сравнение цен между площадками
- ✅ Telegram бот с удобным интерфейсом
- ✅ Показывает выгоду в рублях и процентах

## Установка

```bash
pip install -r requirements.txt
```

**Требования**: Python 3.11+, Chrome браузер

## Настройка

Создайте файл `config.txt` в корне проекта:

```
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_user_id_here
```

### Как получить токен бота:

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

### Как узнать свой Chat ID:

1. Найдите бота @userinfobot в Telegram
2. Отправьте ему любое сообщение
3. Он вернет ваш User ID

## Запуск

```bash
python wb_ozon_bot.py
```

## Использование

1. Отправьте боту команду `/start`
2. Нажмите кнопку "🔍 Найти аналог" или отправьте артикул товара WB
3. Бот найдет товар на Wildberries
4. Бот поищет аналоги на Ozon
5. Получите результат с сравнением цен

### Пример

Отправьте артикул: `15062891`

Бот ответит:
- Информацию о товаре на WB (название, цена, рейтинг)
- Самый дешевый аналог на Ozon
- Разницу в цене (выгоду)

## Архитектура

### WB Parser (`src/parsers/wb_parser.py`)
- Использует публичное API Wildberries (`card.wb.ru`)
- Резервный режим через Selenium если API недоступно
- Возвращает: название, бренд, цену, скидку, рейтинг, отзывы

### Ozon Analog Finder (`src/parsers/ozon_analog_finder.py`)
- Ищет товары на Ozon по поисковому запросу
- Использует существующий парсер `product_parser.py`
- Вычисляет схожесть товаров по названию
- Сортирует результаты по цене и релевантности

### Telegram Bot (`wb_ozon_bot.py`)
- Асинхронный бот на aiogram 3.x
- Запускается в отдельном потоке
- Обрабатывает команды и callback запросы
- Отправляет результаты с форматированием

## Troubleshooting

**Бот не запускается:**
- Проверьте токен в `config.txt`
- Убедитесь что все зависимости установлены

**Товар WB не найден:**
- Проверьте корректность артикула
- Артикул должен содержать только цифры

**Аналоги на Ozon не находятся:**
- Товар может быть уникальным для WB
- Попробуйте другой артикул

**Блокировка Ozon/WB:**
- Используйте прокси
- Увеличьте задержки между запросами
- Запустите в не-headless режиме
